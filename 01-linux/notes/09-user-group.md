sudo useradd newuser (to create newuser)
sudo passwd newuser (to create a password for newuser)
su - newuser (to change to newuser)
exit (to come out of newuser)
sudo usermod -aG sudo newuser (to give sudo permissions to newuser and add to the sudo group)
sudo deluser newuser sudo (to remove newuser from sudo group)
sudo apt-get update (get updates using sudo. can use as a test to see if newuser has sudo permissions) also can use sudo ls /root
sudo groupadd Devops (adds new group called Devops)
sudo groupdel Devops (delete group Devops)
sudo usermod -aG Devops newuser (adds newuser to Devops group)
sudo gpasswd -d newuser Devops (deletes newuser from Devops group)
sudo usermod -aG admin,admin2 newuser (add user to 2 groups)
