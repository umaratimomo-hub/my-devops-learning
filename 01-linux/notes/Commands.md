tab - to complete a name possibility (tab x2 to give all options of that possible file/folder name)

awk - prints specific info from a file ( awk '{print $1}' example.txt prints the first column info from the example.txt)

ps - list processes

kill - stop a process

top - monitor system tasks

df - disk usage

du - directory/file size

man - [command] (gives a manual all about that command)

ls -a (shows hidden directories which start with .)

pwd - (prints current path of directory)

ls - (shows you what's inside the current directory)

ls - [directory name] (shows what’s inside a file in the current directory)

cd - [directory name] (goes to that directory)

cd .. - (goes back one directory)

cat file.txt - (shows whats written in file.txt)

echo “Hello World” > file.txt - (This writes the words Hello World into the file.txt.   > overwrites the file with the new text. >> adds to the file.)

grep “Hello” file.txt - (This finds and highlights the instance of Hello along with showing the rest of the line, inside file.txt)
